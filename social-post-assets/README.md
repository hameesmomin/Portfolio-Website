# Social Post Assets

Use this folder for polished marketing exports for LinkedIn, Instagram posts, carousel covers, story covers, and reel covers.

Create one asset set per product:

- Aura Command: WhatsApp revenue command center
- SecureOps: SIEM-ready cybersecurity and compliance operations
- Documind: UAE document intelligence and renewal tracking
- Siteflow: construction operations and site reporting

Each asset should include the product name, short tagline, 3-5 feature badges, and a clean dashboard preview from sanitized demo data.
